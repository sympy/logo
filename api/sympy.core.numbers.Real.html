Class s.c.n.Real(Number):

Part of sympy.core.numbers

Represents a floating point number. It is capable of representing arbitrary-precision floating-point numbers

Usage:

Real(3.5) .... 3.5 (the 3.5 was converted from a python float) Real("3.0000000000000005")

Notes:

  • Real(x) with x being a Python int/long will return Integer(x)
