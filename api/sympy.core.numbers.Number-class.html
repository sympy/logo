Known Subclasses:
Infinity, Rational, Real

Represents any kind of number in sympy.

Floating point numbers are represented by the Real class. Integer numbers (of any size), together with rational numbers (again, there is no limit on their size) are represented by the Rational class.

If you want to represent for example 1+sqrt(2), then you need to do:

Rational(1) + Rational(2)**( Rational(1)/2 )
