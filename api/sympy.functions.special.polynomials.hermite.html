Class s.f.s.p.hermite(PolynomialSequence):

Part of sympy.functions.special.polynomials

hermite(n, x) gives the nth Hermite polynomial in x, H_n(x)

The Hermite polynomials are orthogonal on (-oo, oo) with respect to the weight exp(-x**2/2).

Examples

>>> x = Symbol('x')
>>> hermite(0, x)
1
>>> hermite(1, x)
2*x
>>> hermite(2, x)
(-2) + 4*x**2
