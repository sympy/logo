getab(self)

source code 

Pretend that self=a*b and return a,b

in general, self=a*b*c*d*..., but in many algorithms, we want to have just 2 arguments to Mul. Use this function to simulate this interface. (the returned b = b*c*d.... )

series(self, sym, n)

source code 
expansion for Mul need to handle this correctly: (e^x-1)/x -> 1+x/2+x^2/6 the first term is (e^x-1)/x evaluated at x=0. normally, we would use a limit. but in cas, the limit is computed using series. so we must calculate series differently - using the bottom up approach: first expand x, then e^x, then e^x-1, and finally (e^x-1)/x
