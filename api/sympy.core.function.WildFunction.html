Helper method for match() - switches the pattern and expr.

Can be used to solve linear equations:

>>> from sympy import Symbol, Wild
>>> a,b = map(Symbol, 'ab')
>>> x = Wild('x')
>>> (a+b*x).matches(0)
{x_: -a/b}
