tangent_line(self, p)

If p is on the ellipse, returns the tangent line through point p. Otherwise, returns the tangent line(s) from p to the ellipse, or None if no tangent line is possible (e.g., p inside ellipse).

_do_line_intersection(self, o)

Find the intersection of a LinearEntity and the ellipse. Makes no regards to what the LinearEntity is because it assumes a Line. To ensure correct intersection results one must invoke intersection() to remove bad results.
