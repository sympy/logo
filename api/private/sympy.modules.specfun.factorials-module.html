binomial Calculate the binomial coefficient C(n,k) = n!/(k!(n-k)!)

  factorial_quotient(p, q)
Calculate the quotient p!/q!, simplifying symbolically if possible.

Function Details

factorial_quotient(p, q)

Usage

Calculate the quotient p!/q!, simplifying symbolically if possible. If both p! and q! are infinite, the correct limit is returned

Examples

>>> from sympy import *
>>> from sympy.modules.specfun.factorials import *
>>> factorial_quotient(pi+1, pi)
1+pi
