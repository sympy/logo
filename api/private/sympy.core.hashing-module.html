Module sympy.core.hashing

This module provides hashing.

usage: 
>>> h = mhash()
>>> h.addint(15)
>>> h.addstr("hey man!")
>>> print h.value
322055065595394571600238891420261570520
mhash.value contains the hash value which is generally a long int, that depends on the order of objects added.

Mbernstein An algorithm produced by Professor Daniel J.
Mmd5 MD5 algorithm, this works fine (so far :).
