        • sympy.core.functions.abs_: Return the absolute value of x

        • sympy.core.functions.exp: Return e raised to the power of x
        • sympy.modules.specfun.factorials.factorial: factorial(x) -> Returns the factorial of x, defined as x! = 1*2*3*...*x if x is a positive integer
        • sympy.modules.specfun.factorials.factorial2: factorial2(x) -> Returns the double factorial of x, defined as x!! = 2*4*6*...*x if x is a positive even integer and as x!! = 1*3*5*...*x if x is a positive odd integer.
        • sympy.modules.specfun.factorials.Function2
          • sympy.modules.specfun.factorials.binomial: Calculate the binomial coefficient C(n,k) = n!/(k!(n-k)!)
          • sympy.modules.specfun.factorials.falling_factorial: Calculate the falling factorial (x)_(n) = x(x-1)...(x-n+1) as a quotient of factorials
          • sympy.modules.specfun.orthogonal_polynomials.legendre: legendre(n, x) - nth Legendre polynomial of x, P_n(x)
          • sympy.modules.specfun.orthogonal_polynomials.legendre_zero: legendre_zero(n, k) represents the kth zero (counting from zero) of the nth Legendre polynomial; that is, if 0 <= k < n, legendre(n, legendre_zero(n, k)) == 0.
          • sympy.modules.specfun.factorials.rising_factorial: Calculate the rising factorial (x)^(n) = x(x+1)...(x+n-1) as a quotient of factorials
        • sympy.core.functions.log: Return the natural logarithm (base e) of x
        • sympy.core.functions.sign
        • sympy.modules.trigonometric.tan: tan(x) -> Returns the tangent of x (measured in radians)
