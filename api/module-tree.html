Module Hierarchy

  • sympy: Sympy is a symbolic manipulation package, written in pure Python.

      • sympy.core.hashing: This module provides hashing.
      • sympy.core.numbers
      • sympy.core.power
      • sympy.core.symbol
    • sympy.modules: Modules of sympy.
      • sympy.modules.decorator
      • sympy.modules.integrals
      • sympy.modules.limits: Implemented according to the PhD thesis http://www.cybertester.com/data/gruntz.pdf, which contains very thorough descriptions of the algorithm including many examples.
      • sympy.modules.mathml: Module with some functions for MathML, like transforming MathML content in MathML presentation.
      • sympy.modules.matrices
      • sympy.modules.paulialgebra
      • sympy.modules.pkgdata: pkgdata is a simple, extensible way for a package to acquire data file resources.
      • sympy.modules.polynomials: Module with some routines for polynomials
      • sympy.modules.printing: Module for printing.
        • sympy.modules.printing.gtk: Printing using GtkMathView
        • sympy.modules.printing.latex: Some utility functions to convert sympy's objects to latex code
        • sympy.modules.printing.pretty: Prettyprinter idea by Jurjen Bos 2007
