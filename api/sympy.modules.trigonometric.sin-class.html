Usage

sin(x) -> Returns the sine of x (measured in radians)

Notes

sin(x) will evaluate automatically in the case x is a multiple of pi.

Further examples

>>> from sympy import *
>>> x = Symbol('x')
>>> sin(x**2).diff(x)
2*cos(x^2)*x
>>> sin(1).diff(x)
0
>>> sin(pi)
0
