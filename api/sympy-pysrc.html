 1  """ 
 2  Computer Algebra System (CAS) in Python 
 3  ======================================= 
 4   
 5  Sympy is a symbolic manipulation package, written in pure Python. Its aim is to 
 6  become a full featured CAS in Python, while keeping the code as simple as 
 7  possible in order to be comprehensible and easily extensible. 
 8   
 9  Features 
10  ======== 
11   
12  Currently, SymPy core has only around 1500 lines of code (including extensive 
13  comments) and its capabilities include: 
14   
15      - basic arithmetics *,/,+,- 
16      - basic simplification (like a*b*b + 2*b*a*b -> 3*a*b^2) 
17      - expansion (like (a+b)^2 -> a^2 + 2*a*b + b^2) 
18      - functions (exp, ln, sin, cos, tan, ...) 
19      - complex numbers (like exp(I*x).evalc() -> 
20      - cos(x)+I*sin(x)) 
21      - differentiation 
22      - taylor series 
23      - basic substitution (like x-> ln(x)) 
24      - arbitrary precision integers, 
25      - rationals and floats  
26   
27  Then there are SymPy modules (1000 lines) for these tasks: 
28   
29      - limits (like limit(x*log(x), x, 0) -> 0) 
30      - integration (currently it can only do very simple integrals) 
31      - polynomials (division, gcd, square free decomposition) 
32      - symbolic matrices  
33   
34  To help you get started, here is a simple example in the python 
35  interpreter: 
36   
37      >>> from sympy import Symbol, cos 
38      >>> x = Symbol('x') 
39      >>> e = 1/cos(x) 
40      >>> print e.series(x,10) 
41      1+50521/3628800*x**10+61/720*x**6+1/2*x**2+5/24*x**4+277/8064*x**8 
42   
43  For full documentation, see the docs at our web page: 
44  U{http://code.google.com/p/sympy/wiki/Documentation} 
45   
46  Structure 
47  ========= 
48   
49  Sympy is basically divided in two modules: the core module, which contains  
50  the classes needed for basic symbolic manipulations, like the definition of a 
51  Symbol or a Rational number, and sympy.modules, a high-level module that  
52  contains algorithms for symbolic computations, like limit, integration, etc. 
53   
54   
55  G{packagetree sympy} 
56  """
