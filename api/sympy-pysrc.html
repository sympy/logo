 1  """ 
 2  Computer Algebra System (CAS) in Python 
 3  ======================================= 
 4   
 5  Sympy is a symbolic manipulation package, written in pure Python. Its aim is to 
 6  become a full featured CAS in Python, while keeping the code as simple as 
 7  possible in order to be comprehensible and easily extensible. 
 8   
 9  Features 
10  ======== 
11   
12  Currently, Sympy has only around 1600 lines of code (including comments) and 
13  its capabilities include: 
14   
15      - basic arithmetics *, /, +, - 
16      - basic simplification (like a*b*b + 2*b*a*b -> 3*a*b^2) 
17      - expansion (like (a+b)^2 -> a^2 + 2*a*b + b^2) 
18      - functions (exp, ln, sin, cos, tan, ...) 
19      - differentiation 
20      - integration (currently it can do only very simple integrals) 
21      - taylor series 
22      - basic substitution (like x-> ln(x)) 
23      - arbitrary precision integers, rationals and floats 
24      - complex numbers (currently only simple things) 
25      - limits 
26   
27  To help you get started, here is a simple example in the python 
28  interpreter: 
29   
30      >>> from sympy import Symbol, cos 
31      >>> x = Symbol('x') 
32      >>> e = 1/cos(x) 
33      >>> print e.series(x,10) 
34      1+50521/3628800*x**10+61/720*x**6+1/2*x**2+5/24*x**4+277/8064*x**8 
35   
36  For full documentation, see the docs at our web page: 
37  U{http://code.google.com/p/sympy/wiki/Documentation} 
38   
39  Structure 
40  ========= 
41   
42  Sympy is basically divided in two modules: the core module, which contains  
43  the classes needed for basic symbolic manipulations, like the definition of a 
44  Symbol or a Rational number, and sympy.modules, a high-level module that  
45  contains algorithms for symbolic computations, like limit, integration, etc. 
46   
47   
48  G{packagetree sympy} 
49  """
