 1  """ 
 2  Computer Algebra System (CAS) in Python 
 3  ======================================= 
 4   
 5  Sympy is a symbolic manipulation package, written in pure Python. Its aim is to 
 6  become a full featured CAS in Python, while keeping the code as simple as 
 7  possible in order to be comprehensible and easily extensible. 
 8   
 9  Features 
10  ======== 
11   
12  Currently, Sympy has only around 1600 lines of code (including comments) and 
13  its capabilities include: 
14   
15      - basic arithmetics *, /, +, - 
16      - basic simplification (like a*b*b + 2*b*a*b -> 3*a*b^2) 
17      - expansion (like (a+b)^2 -> a^2 + 2*a*b + b^2) 
18      - functions (exp, ln, sin, cos, tan, ...) 
19      - differentiation 
20      - integration (currently it can do only very simple integrals) 
21      - taylor series 
22      - basic substitution (like x-> ln(x)) 
23      - arbitrary precision integers and 
24      - rationals 
25      - standard (python) floats 
26      - complex numbers (currently only simple things) 
27      - limits 
28   
29  To help you get started, here is a simple example in the python 
30  interpreter: 
31   
32      >>> from sympy import Symbol, cos 
33      >>> x = Symbol('x') 
34      >>> e = 1/cos(x) 
35      >>> print e.series(x,10) 
36      1+50521/3628800*x^10+61/720*x^6+1/2*x^2+5/24*x^4+277/8064*x^8 
37   
38  For full documentation, see the docs at our web page: 
39  U{http://code.google.com/p/sympy/wiki/Documentation} 
40   
41  Structure 
42  ========= 
43   
44  Sympy is basically divided in two modules: the core module, which contains the classes 
45  needed for basic symbolic manipulations, like the definition of Symbol or Rational number, and 
46  a some high-level modules, that are responsable for high-level work done with sympy:  
47  limits, integration, solving, etc., and is the one you will have to look at if you are 
48  searching for examples on how to extend sympy 
49   
50   
51  G{packagetree sympy} 
52  """
