def arg(x):

Returns the complex argument (phase) of x. The returned value is an mpf instance. The argument is here defined to satisfy -pi < arg(x) <= pi. On the negative real half-axis, it is taken to be +pi.

def log(x, b=None):

Returns the base-b logarithm of x. If b is unspecified, return the natural (base-e) logarithm. log(x, b) is defined as log(x)/log(b). log(0) raises ValueError.

The natural logarithm is real if x > 0 and complex if x < 0 or if x is complex. The principal branch of the complex logarithm is chosen, for which Im(log(x)) = -pi < arg(x) <= pi.

def acos(x):

Returns the inverse cosine of x. Outside the range [-1, 1], the result is complex and defined as the principal branch value of pi/2 + i * log(i * x + sqrt(1 - x**2)).

def asinh(x):

Returns the inverse hyperbolic sine of x. For complex x, the result is the principal branch value of log(x + sqrt(1 + x**2)).

def acosh(x):

Returns the inverse hyperbolic cosine of x. The value is given by log(x + sqrt(1 + x**2)), where the principal branch is used when the result is complex.

def atanh(x):

Returns the inverse hyperbolic tangent of x. Outside the range [-1, 1], the result is complex and defined as the principal branch value of (log(1+x) - log(1-x))/2.
