Class s.t.m.m.mpc(mpnumeric):

Part of sympy.thirdparty.mpmath.mpmath

An mpc represents a complex number using a pair of mpf:s (one for the real part and another for the imaginary part.) The mpc class behaves fairly similarly to Python's complex type.
