Class s.t.m.m.constant(mpf):

Part of sympy.thirdparty.mpmath.mpmath

Represents a mathematical constant with dynamic precision. When printed or used in an arithmetic operation, a constant is converted to a regular mpf at the working precision. A regular mpf can also be obtained using the operation +x.

A new mpf can be created from a Python float, an int, a
Decimal, or a decimal string representing a number in
floating-point format. Examples:

    mpf(25)
    mpf(2.5)
    mpf('2.5')
    mpf('1.6e1000')

An mpf can also be created from a tuple (man, exp) or
(man, exp, bc):

    >>> mpf((3, -1))
    1.5
