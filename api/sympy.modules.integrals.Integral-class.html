                  Integral

Definite integral.

Integral(f, (x,a,b) represents \int_a^b f(x) dx

Usage

print Integral(1/t, (t,1,x)) will print: 
    int_{1}^{x} (t^(-1)) dt
print Integral(1/t, (t,1,x)).doit() will print: 
   log(x)
print Integral(1/t, (t,1,x)).diff(x) will print: 
   1/x
Currently can only integrate very simple functions, like polynoms. You can however implement as many formulas as you want at the end of the primitive_function() function.
