def normalize(man, exp, prec, rounding):

normalize(man, exp, prec, rounding) -> return tuple representing a fully rounded and normalized raw mpf with value (man * 2**exp)

The mantissa is rounded in the specified direction if the number of exceeds the precision. Trailing zero bits are also stripped from the mantissa to ensure that the representation is canonical.
