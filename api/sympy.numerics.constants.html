Module s.n.constants

Part of sympy.numerics

Commonly used mathematical constants (pi, log(2), log(10), gamma, ...)

There are two functions for each constant: const_fixed(prec) returns a big integer n such that const ~= n * 2**-prec, and const_float() returns Float(const), rounded to the current Float working precision
