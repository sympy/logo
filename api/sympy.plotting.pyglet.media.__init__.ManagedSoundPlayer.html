Class s.p.p.m._.ManagedSoundPlayer(Player):

Part of sympy.plotting.pyglet.media.__init__

A player which takes care of updating its own audio buffers.

This player will continue playing the sound until the sound is finished, even if the application discards the player early. There is no need to call `Player.dispatch_events` on this player, though you must call `pyglet.media.dispatch_events`.
