def npartitions(n):

Calculate the partition function P(n), i.e. the number of ways that n can be written as a sum of positive integers.

P(n) is computed using a straightforward implementation of the Hardy-Ramanujan-Rademacher formula, described e.g. at http://mathworld.wolfram.com/PartitionFunctionP.html

The speed is decent up to n = 10**5 or so. The function has been tested to give the correct result for n = 10**6.
