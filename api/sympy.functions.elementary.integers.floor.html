Class s.f.e.i.floor(Function):

Part of sympy.functions.elementary.integers

Floor is a univariate function which returns the largest integer value not greater than its argument. However this implementaion generalizes floor to complex numbers.

More information can be found in "Concrete mathematics" by Graham, pp. 87 or visit http://mathworld.wolfram.com/FloorFunction.html. 
>>> from sympy import *
>>> floor(17)
17
>>> floor(Rational(23, 10))
2
>>> floor(2*E)
5
>>> floor(-Real(0.567))
-1
>>> floor(-I/2)
-I

def canonize(cls, arg):
Returns a canonical form of cls applied to arguments args.

The canonize() method is called when the class cls is about to be
instantiated and it should return either some simplified instance
(possible of some other class), or if the class cls should be
unmodified, return None.

Example of canonize() for the function "sign"
---------------------------------------------

@classmethod
def canonize(cls, arg):
    if isinstance(arg, Basic.NaN):
        return S.NaN
    if isinstance(arg, Basic.Zero): return S.One
    if arg.is_positive: return S.One
    if arg.is_negative: return S.NegativeOne
    if isinstance(arg, Basic.Mul):
        coeff, terms = arg.as_coeff_terms()
        if not isinstance(coeff, Basic.One):
            return cls(coeff) * cls(Basic.Mul(*terms))
