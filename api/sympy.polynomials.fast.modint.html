def xgcd(a, b):
Returns g, x, y such that g = x*a + y*b = gcd(a,b).
Input:
    a -- an integer
    b -- an integer
Output:
    g -- an integer, the gcd of a and b
    x -- an integer
    y -- an integer
Examples:
>>> xgcd(2,3)
(1, -1, 1)
>>> xgcd(10, 12)
(2, -1, 1)
>>> g, x, y = xgcd(100, 2004)
>>> print g, x, y
4 -20 1
>>> print x*100 + y*2004
4
def crt(m, v, symmetric=False):

Chinese remainder theorem.

The integers in m are assumed to be pairwise coprime. The output is then an integer f, such that that f = v_i mod m_i for each pair out of v and m.
