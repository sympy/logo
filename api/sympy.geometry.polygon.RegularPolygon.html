Class s.g.p.RegularPolygon(Polygon):

Part of sympy.geometry.polygon

A polygon which is regular. Such a polygon has all internal angles equal and all sides the same length.

Usage:

The polygon is constructed from a center point, a radius value, and the number of sides. The radius value is the distance from the center to a vertex.

Example:

>>> RegularPolygon(Point(0, 0), 5, 5)
RegularPolygon(Point(0, 0), 5, 5)

Examples:
======
    >>> from sympy.geometry import *
    >>> p1,p2,p3,p4 = map(Point, [(0,0), (1,0), (5,1), (0,1)])
    >>> poly = Polygon(p1, p2, p3, p4)
    >>> poly.angles[p1]
    (1/2)*pi
    >>> poly.angles[p2]
    acos(-4/17*17**(1/2))
