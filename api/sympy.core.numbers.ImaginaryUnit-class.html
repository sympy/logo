evalf(self)

source code 
Evaluate to a float. By convention, will return 0, which means that evalf() of a complex number will mean the projection of the complex plane to the real line. For example: >>> (1-2*I).evalf() 1.00 >>> (-2+1*I).evalf() (-2.0)
