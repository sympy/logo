def polyfunc(expr, derivative=False):
Convert a SymPy expression representing a univariate polynomial into a function for numerical evaluation using Floats / ComplexFloats. 
>>> x = Symbol('x')
>>> polyfunc(x**3 + 4)(2)
Float('12')
If derivative=True is set, the evaluation function evaluates both the polynomial and its derivative at the given point and returns the two values as a tuple.
