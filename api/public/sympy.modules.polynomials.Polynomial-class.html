Class Polynomial

Polynomial representation in coefficient list form.

This brings higher efficiency, but also more readable code, since there is no more need of two representation side-by-side that need to be in sync, in all the internal algorithms.

Examples: >>> x = Symbol('x') >>> y = Symbol('y') >>> f = Polynomial(x) >>> f Polynomial(x, [x], 'grevlex', 'int') >>> g = Polynomial(y+1) >>> f + g Polynomial(1+x+y, [x, y], 'grevlex', 'int') >>> f * g Polynomial(x+x*y, [x, y], 'grevlex', 'int')
