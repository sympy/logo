Class Ideal

Describes a polynomial ideal over the real numbers.

Try to avoid different variables and orders between the ideals, give them explicitly; the automatic handlers don't always act as expected.

Examples: >>> x = Symbol('x') >>> y = Symbol('y') >>> I = Ideal([x, y**2]) >>> J = Ideal(x*y) >>> I == J False >>> I*J == I.intersect(J) False >>> I**2 == I*I True
