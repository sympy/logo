Class s.s.d.Normal(ContinuousProbability):

Part of sympy.statistics.distributions

Normal(mu, sigma) represents the normal or Gaussian distribution with mean value mu and standard deviation sigma.

Example usage: 
>>> N = Normal(1, 2)
>>> N.mean
1
>>> N.variance
4
>>> N.probability(-oo, 1)   # probability on an interval
1/2
>>> N.probability(1, oo)
1/2
>>> N.probability(-oo, oo)
1
>>> N.probability(-1, 3)
erf((1/2)**(1/2))
>>> _.evalf()
0.682689492137086

def confidence(s, p):
Return a symmetric (p*100)% confidence interval. For example,
p=0.95 gives a 95% confidence interval. Currently this function
only handles numerical values except in the trivial case p=1.

Examples usage:
    # One standard deviation
    >>> N = Normal(0, 1)
    >>> N.confidence(0.68)
    (-0.994457883209753, 0.994457883209753)
    >>> N.probability(*_).evalf()
    0.68

    # Two standard deviations
    >>> N = Normal(0, 1)
    >>> N.confidence(0.95)
    (-1.95996398454005, 1.95996398454005)
    >>> N.probability(*_).evalf()
    0.95
