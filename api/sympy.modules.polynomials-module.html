get_poly(p, x)

source code 

Returns a python list representing the polynomial 'p(x)'.

'p' is a polynomial, for example: x**2 + 3*x*y.sqrt() - 8 'x' is the variable of the polynomial, for example: x get_poly returns a python list of the form [(coeff0,n0), (coeff1,n1), ...] where p = coeff0*x**n0 + coeff1*x**n1 + ... and n0, n1, ... are sorted from the lower exponents up.

Example: >>> from sympy import * >>> from sympy.modules.polynomials import get_poly >>> x = Symbol("x") >>> y = Symbol("y") >>> get_poly(x**2 + 3*x**7*y.sqrt() - 8, x) [(-8, 0), (1, 2), (3*y**(1/2), 7)]

div(f, g, x)

source code 

Expresses f = g*q + r, returns (q,r)

All coefficients of 'f' and 'g' are assumed to be integers, and coefficients in 'q' and 'r' are then guaranteed to be integers.
