right(self, *args)

source code 
Put pictures next to this one. (Multiline) strings are allowed, and are given a baseline of 0. 
>>> print StringPict("10").right("+",StringPict("1
-
2",1))
   1
10+-
   2

stack(*args)
Static Method

source code 
Put pictures on top of each other, from top to bottom. The baseline is the baseline of the second picture. Everything is centered. Baseline is the baseline of the second picture. Strings are allowed. The special value StringPict.LINE is a row of '-' extended to the width.

below(self, *args)

source code 
Put pictures under this picture. Baseline is baseline of top picture: 
       >>> print StringPict("x+3").below(StringPict.LINE, '3')
       x+3
--- 3
